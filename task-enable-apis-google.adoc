= Habilite las API de Google Cloud
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Se necesitan varias API para poner en marcha Connector y Cloud Volumes ONTAP en Google Cloud.

.Paso
. Habilite las siguientes API de Google Cloud en su proyecto:
+
** API de Cloud Deployment Manager V2
** API de registro en la nube
** API de Cloud Resource Manager
** API del motor de computación
** API de gestión de acceso e identidad (IAM)
